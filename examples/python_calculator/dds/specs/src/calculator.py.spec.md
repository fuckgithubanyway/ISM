# Спецификация Артефакта: src/calculator.py

## Описание

Этот артефакт представляет собой модуль Python с базовыми арифметическими функциями.

## Требования к реализации

Модуль должен содержать четыре функции:

1.  **`add(a: float, b: float) -> float`**
    - Назначение: Сложение двух чисел.
    - Возвращает сумму `a` и `b`.

2.  **`subtract(a: float, b: float) -> float`**
    - Назначение: Вычитание одного числа из другого.
    - Возвращает разность `a` и `b`.

3.  **`multiply(a: float, b: float) -> float`**
    - Назначение: Умножение двух чисел.
    - Возвращает произведение `a` и `b`.

4.  **`divide(a: float, b: float) -> float`**
    - Назначение: Деление одного числа на другое.
    - Возвращает результат деления `a` на `b`.
    - **Обработка ошибок:** Если аргумент `b` равен нулю, функция должна вызывать исключение `ValueError` с сообщением "Division by zero is not allowed.".
