# Демонстрационный проект: Калькулятор на Python

Этот проект демонстрирует использование парадигмы DDS для создания простого модуля-калькулятора.

## Структура

- `dds/specs/_project.spec.md`: Определяет общие правила (Python, PEP8, docstrings).
- `dds/specs/src/calculator.py.spec.md`: Описывает требуемые функции модуля `calculator.py`.
- `src/`: Директория, в которой Исполнитель (LLM) должен будет создать артефакт `calculator.py`.

## Как запустить DDS-цикл

Для генерации артефакта `src/calculator.py` выполните следующие шаги, находясь в этой директории (`examples/python_calculator`).

### Шаг 1: Сгенерируйте Снимок

Выполните команду ниже. Она соберет все спецификации в один текстовый блок и сохранит его в файл `snapshot.txt`.

```bash
./dds/scripts/dds-snapshot ./ > snapshot.txt
