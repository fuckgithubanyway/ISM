# PDS Manifest: Topology & Artifacts

**Версия:** 0.2
**Язык:** Русский


## Система Артефактов (PDS Quadrivium)

В Зоне Определения (`[root]/pds/`) допускаются только четыре типа файлов.

### Манифест (`-manifest-[name].md`)
*   **Локация:** Только корень `[root]/pds/`.
*   **Роль:** Определение методологии и глобальных законов проекта.

### Мета-Спецификация (`-meta-[name].md`)
*   **Локация:** Любая директория.
*   **Роль:** Контекстные ограничения (Constraints) для текущей директории и поддиректорий.
*   **Пример:** Стек технологий, Code Style, паттерны проектирования, запрещенные функции.
*   **Иерархическое Наследование:** Правила Мета-спецификаций распространяются на текущую директорию и все вложенные поддиректории. В случае конфликта приоритет имеет спецификация с максимальной вложенностью (ближайшая к целевому файлу).
*@ADR: Каскадное наследование мета-спецификаций позволяет определять локальные технологические исключения (например, особый стек для микросервиса) без усложнения глобальных правил проекта.

### Функциональная Спецификация (`-spec-[name].md`)
*   **Локация:** Изоморфно целевому модулю.
*   **Роль:** Описание логики ("Смысл").
*   **Отношение:** 1 к N (одна спека может описывать логический модуль из нескольких файлов).

### Спецификация Реализации (`-spec-[name].[ext]`)
*   **Локация:** Изоморфно целевому файлу (1 к 1).
*   **Формат:** Исходный код на целевом языке (без суффикса `.md`).
*   **Роль:** Технический скелет ("Физика").
*   **Содержание:** Сигнатуры, импорты, жестко заданные структуры. Место для ручного кода, который Агент не имеет права менять.
*   **Принцип Неизменного Скелета (Immutable Skeleton):** Любой фрагмент кода или структура, описанная в Спецификации Реализации, имеет статус `Immutable`. Агент обязан перенести этот фрагмент в Проекцию дословно. Модификация этих участков Агентом запрещена, кроме случаев явной команды в режиме Реконсиляции.
*@ADR: Введение неизменного скелета (`-spec-[name].[ext]`) является необходимым ограничением недетерминированности интеллектуального Агента для гарантированного соблюдения типов и интерфейсных контрактов.
